# STM32F407ZET6 chinese board

a) stm32f407zet6  
b) 512K flash  
c) 196K sram  
e) 8Mhz external crystal  

## external SRAM
IS62WV51216BLL  
a) connected to FSMC via Bank1/CS3

C17/C18 should be soldered.

32Pin
